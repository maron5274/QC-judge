## 環境構築
node.jsのインストール
[https://nodejs.org/ja/download/](https://nodejs.org/ja/download/)

このレポジトリをクローン
```bash
git clone https://github.com/maron5274/QC-judge.git
```

node_modulesのインストール
```bash
npm install
```

## ローカル開発
```bash
npm run dev
```
